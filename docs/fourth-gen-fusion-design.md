# Fourth-Gen Fusion Design Summary

The clean rebuild keeps Hstar API compatibility as the product foundation, absorbs MovoAi interaction ideas as reference, and treats FrameAI as a future node capability reference.

The first MVP is intentionally thin but real: a user can create a prompt node, connect it to an image generation node, run a provider-backed task, persist the output as an asset, save the workflow, and reopen it without data loss.

All frontend UI interaction text must be Simplified Chinese and UTF-8 encoded.
